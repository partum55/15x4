# 15x4

15x4 -- це відкритий науково-популярний лекторій, де четверо спікерів за 15 хвилин кожен розповідають про те, що їх по-справжньому захоплює. Формат народився в Харкові і швидко розлетівся містами України та Східної Європи. Наша мета проста: зробити науку ближчою до кожного -- через короткі, живі та зрозумілі лекції.

Цей вебсайт -- обличчя проєкту в інтернеті. Тут можна дізнатися про найближчі події, переглянути архів лекцій, познайомитися з нашою командою та приєднатися як спікер чи організатор.

---

## Run the website locally

This project uses Vite + React + TypeScript.

### Prerequisites

- Node.js 18+ (LTS recommended)
- npm

### Start in development mode

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal (usually `http://localhost:5173`).

### Build and preview (production-like)

```bash
npm run build
npm run preview
```

Use this mode to verify that all implemented features work in the built version.

### Optional quality checks

```bash
npm run lint
```
